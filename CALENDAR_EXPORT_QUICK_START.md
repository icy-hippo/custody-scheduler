## â“ Troubleshooting

### File Won't Download
- Check your browser's download settings
- Try a different browser
- Try copying to clipboard instead

### Calendar App Won't Accept File
- Verify it's a .ics file
- Try importing in web version of the app instead
- Check file isn't corrupted (re-download)
- Ensure you have editing permissions on the calendar

### Events Don't Show Up
- Make sure the calendar is enabled/visible in your app
- Check the date range (our exports start from today forward)
- Verify the time zone is correct
- Try refreshing your calendar app

### Missing Details
- Some calendar apps don't display all fields
- Check the event details in your calendar app
- Details like notes are usually in the event description
